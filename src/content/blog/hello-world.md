This is my dev journal. I'll be writing about frontend architecture,
TypeScript patterns, and whatever I'm currently building.

## Why Astro?

Fast, static, and stays out of the way. Astro's island architecture means
I get interactive React components where I need them, and zero JavaScript
where I don't. Perfect for a content-focused blog.

## The Stack

- **Astro** for the framework
- **React** for interactive islands (search, theme toggle)
- **Tailwind CSS v4** for styling
- **Pagefind** for static search
- **GitHub Pages** for hosting

Simple, fast, and easy to maintain.
